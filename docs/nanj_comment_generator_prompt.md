# リアルタイムプロ野球なんJ語コメント生成プロンプト

## 基本指示

**指定日付の最新プロ野球情報から、指定3サイトのなんJコメントを抽出・参考にして60個のコメント（各独立テーマ20個ずつ）を生成してください。**

## ステップ1: 情報取得

### 1.1 対象サイトの確認（必須）
以下の**3つのサイトを必ず確認**し、共通して取り上げられている話題を特定：

```
1. http://blog.livedoor.jp/nanjstu/
   - なんJスタジアム@なんJまとめ
   - 野球の実況まとめ・コメント抽出

2. http://blog.livedoor.jp/yakiusoku/
   - やきう速報
   - プロ野球速報・まとめ・コメント収集

3. https://nanjpride.blog.jp/
   - なんJ PRIDE
   - 野球関連のなんJ語コメント・反応まとめ
```

### 1.2 プロ野球情報の取得
```
対象日付: [指定された日付] (今日/昨日/2025-09-10等)
基本情報: NPB公式、スポーツナビ、各球団公式等
収集内容: 試合結果、スコア、選手名、記録、球場等
```

### 1.3 実際のコメント抽出
指定3サイトから**実際に投稿されているコメント**を確認：
```
- 記事のコメント欄
- まとめられた実況コメント
- ユーザーの反応・感想
- 話題になっている内容の特定
```

## ステップ2: 独立テーマ選定（3つ）

**3つのサイトで取り上げられている話題**から3つの独立したテーマを選定：

### テーマ選定基準
1. **各サイトで言及されている話題**から多様性を重視
2. **実際にコメントが多く投稿されている内容**  
3. **感情的反応が大きく、なんJ語で表現されている出来事**
4. **3つのテーマは互いに独立し、重複しない内容**

### 選定手順
```
Step A: 各サイトの主要話題をリスト化
Step B: 話題の多様性と独立性を確認
Step C: コメント数・反応の大きさで評価
Step D: 重複のない独立した3テーマを選定
```

### 独立テーマの例
- テーマ1: 試合の展開・結果（劇的な逆転勝利・接戦等）
- テーマ2: 選手の個人成績・活躍（記録達成・好投等）
- テーマ3: 特別な出来事・話題（珍事・監督采配・ファンの反応等）

**※各テーマは独立しており、重複する内容は避ける**

## ステップ3: コメント生成（各テーマ20個ずつ）

### 生成方針
- **3サイトの実際のコメントを参考**にしつつ、**創作・再構成**して生成
- **直接コピー禁止**: 実在コメントをそのまま使用しない
- **実際の情報ベース**: 正確な選手名・スコア・展開を使用
- **各サイトの特色を反映**: サイト毎のコメント傾向を考慮

### なんJ語の特徴を含める
```
語尾表現: やで、やろ、やん、ンゴ、や
一人称: ワイ
笑い表現: 草、草生える、草不可避  
驚き表現: ファッ！？、マジかよ、やばない？
チーム愛称: 虎（阪神）、鷹（ソフバン）、燕（ヤクルト）等
野球用語: OPS、QS、WAR、敗戦処理等
関西弁調: 〜すぎやろ、〜してもうた、〜やんけ
```

### サイト別の特色を反映
```
nanjstu系: 実況寄りの熱い反応・リアルタイム感
yakiusoku系: 速報性重視・的確な分析コメント
nanjpride系: エンタメ性・面白さ重視のコメント
```

### 感情表現の種類
```
歓喜: やったぜ、すげー、神かよ
怒り: ふざけんな、やめたれw、クソすぎ
困惑: なんやこれ、意味わからん、は？
皮肉: さすがですね、お疲れ様でした
興奮: 鳥肌立った、やべーよ、ガチで
```

## ステップ4: 品質チェック

### 情報の正確性
- ✅ 実際の選手名・チーム名
- ✅ 正確なスコア・記録  
- ✅ 実際の球場名・日付
- ✅ 3サイトでの共通話題確認

### なんJ語の自然性
- ✅ 適度なスラング使用（過度でない）
- ✅ 野球ファンらしい視点
- ✅ 3サイトの特色を反映したリアルなコメント感

### 動画制作適合性  
- ✅ 音声合成適切な長さ（3-8秒程度）
- ✅ 字幕表示適切な文字数（30文字以内推奨）
- ✅ 視聴者理解しやすい表現
